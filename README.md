
# Handwriting to PDF

Project convert Handwriting image to PDF file


## Installation

Install all necessary libs:

```bash
    pip install -r requiment.txt
```

Create Lib folder

```bash
    mkdir venv/Lib
```

Copy your Tesseract-OCR to venv/Lib:
```bash
venv
 └── Lib
      └── Tesseract-OCR
```
